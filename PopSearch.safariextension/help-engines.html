PopSearch Help

Adding search engines

To add a search engine, click the Add Search Engine button at the bottom of the engine table. A new row will appear at the bottom of the table. Enter a name for the new engine, a key letter (optional), and the search URL for the engine. In the search URL, use the characters %s to stand for the search term(s). When you're done, click Save.

Determining a search URL

If you want to add a search engine but you're not sure how to get the URL you need, here's what you do.

  1. Open a browser tab and go to the search page of the search engine you want to add. In most cases, that would just be the site's home page.
  2. Perform a search as you would normally, using some memorable search term such as "dogcow" (without the quotes).
  3. When the results page finishes loading, click in Safari's address bar, select the entire URL, and copy it (type Command-C).
  4. Go back to PopSearch's settings page. In the URL box of the new engine, paste the URL you copied from the address bar.
  5. Find the search term that you entered and replace it with the characters "%s". For example, if the old URL was "http://www.imdb.com/find?s=all&q=dogcow", the new URL would be "http://www.imdb.com/find?s=all&q=%s".
  6. Click Save.

Variables in search URLs

There are three variables that you can use in search URLs. They are as follows.

  • %s: As discussed above, the variable %s stands for the search terms you enter in the PopSearch search input box.
  • %u: The variable %u stands for the full URL of the current web page. Use this whenever you want to pass the current URL as a parameter to the search engine.
  • %h: The variable %h stands for the hostname portion of the current web page's URL. This is useful, for example, when constructing a site-specific search. (See the next topic for more information.)

Site-specific searches

PopSearch can perform site-specific searches; that is, searches limited to the particular website you are currently viewing in Safari. PopSearch comes with two predefined site-specific search engines: Google Here and Bing Here. The way a site-specific search works is that PopSearch appends a special keyword, followed by the hostname of the current site, to the search query. The special keyword tells the search engine to limit the search to the specified site.

You can add your own site-specific searches to PopSearch. To make an engine site-specific, just insert the characters "%h" in the URL where the hostname should go. In general, you can insert "%h" after the keyword that tells the search engine to search a particular site. For example, in the URL of the predefined Google Here engine, the site-specific keyword is "site:", and "%h" follows immediately after.

Keywords

When adding a search engine, you can optionally enter a keyword for that engine. (The keyword is what appears under the "Key" heading in the engine table.) Entering a keyword will enable you to switch to the search engine in the PopSearch popup without using the engine menu: just type the keyword into the search input box, followed by a space, before you type the search terms.

A keyword can contain any number of characters, but no spaces. Try not to use a real word as a keyword, because if you do, you won't be able to search for it!

Using the Engine Wizard

Starting with version 1.2, PopSearch includes a feature called the Engine Wizard that often makes it possible to add a new search engine to PopSearch without going through the settings page. When you start the Engine Wizard on the results page of a search website, the wizard tries to determine the correct search URL for the engine by matching the contents of a non-empty text input box on the page with the search terms in the page URL. If a match is found, the wizard offers to add the engine to PopSearch with the %s token already inserted at the (hopefully) correct location.

To use the Engine Wizard:

  1. Nagivate to the home page of the search engine you want to add.
  2. Perform a search with one or more search terms. Any terms will do, but to be safe, avoid weird characters in the input.
  3. When the search results finish loading, right-click (or control-click) anywhere on the page and select Add This Engine To PopSearch in the context menu. (If the command does not appear in the context menu, try right-clicking in a text input box that contains the search terms.)
  4. Follow the prompts to add the new engine: first, enter a name for the engine, then optionally enter a keyword, and finally, check the URL.

The newly added search engine will be available for use in PopSearch immediately.

Adding bookmarklets

Starting with version 1.2.6, PopSearch supports opening JavaScript bookmarklets via its popup. You may (or may not) find this useful as a keyboard-based alternative to selecting bookmarklets in Safari's bookmarks menu or toolbar. You can open a bookmarklet in PopSearch the same way you open a search engine: just select it in the engine menu or type its keyword followed by a space in the input box, and press Enter. When opening a bookmarklet, PopSearch ignores the contents of the input box and the target menu, and simply opens the bookmarklet in the current tab.

To add a bookmarklet to PopSearch, follow the same steps as when adding a search engine, but in the URL box, paste the bookmarklet's code, making sure it begins with "javascript:".

Backing up your search engines to (and restoring them from) the cloud

If you have an account with either Delicious or Pinboard, you can back up your PopSearch search engines to the cloud bookmarking service, where they will be saved as bookmarks tagged "PopSearch". Once you've done that, if you ever need to recreate your search engine list, either on the same computer or on another, you can do so by simply restoring your PopSearch-tagged bookmarks from the cloud.

The first time you back up your search engines, the bookmarking service will create a bookmark tagged "PopSearch" for each engine. Thereafter, whenever you back up again, the service will add new bookmarks only for those search engines whose URLs are new or changed. Thus, if you change the URL of a search engine but leave its name unchanged, when you back up again, you will have two bookmarks on the service with the same name but different URLs. PopSearch will never delete a bookmark on the service.

Conversely, when you restore your engines from the bookmarking service, PopSearch will add a new search engine for any bookmark whose URL is different from the URLs of all existing search engines. If a bookmark has the same name as an existing search engine but a different URL, PopSearch will add the bookmark as a new search engine with that name, and append "(old)" to the name of the old search engine. PopSearch will never delete a search engine as a result of restoring from your bookmarking service.

Note: The first time you back up your engines to a particular service, Safari will prompt you to enter your username and password for that service. You can choose to let Safari remember this information for you so you don't have to enter it again the next time you back up or restore. If you enter an incorrect username or password, you will need to restart Safari before trying again.
