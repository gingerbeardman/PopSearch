PopSearch Help

Setting the Hotkey

To set the hotkey that invokes the PopSearch popup, click inside the input box and press the desired key combination. The hotkey is saved immediately and can be used right away.

The hotkey can be any character key, with or without Command, Option, Control, Shift, or any combination of those modifiers. By "character key" I mean any key that produces a visible character if you press it while typing text. (This does NOT include special keys such as PageDown, Home, NumLock, the arrow keys, etc.)

Note: Due to a bug, if you enter a hotkey that uses Control as the only modifier, on rare occasions, PopSearch may not notice that the Control key was pressed. If this happens, just enter the same hotkey again, and PopSearch will record it correctly.

Tab Settings

Open search results in new tab

If this option is selected, search results will open in a new tab instead of the current tab.

You can temporarily reverse this setting by holding down the Command key (or, on a Windows computer, the Ctrl key) when executing a search.

Open results in a new tab when searching on selected text

If this option is selected, search results will open in a new tab if you had selected some text on the current page before invoking PopSearch.

You can temporarily reverse this setting by holding down the Command key (or, on a Windows computer, the Ctrl key) when executing a search.

Open new tab in background

If this option is selected, when opening search results in a new tab, the new tab will be opened in the background. Otherwise, the new tab will become active upon opening.

You can temporarily reverse this setting by holding down the Shift key when executing a search.

Position new tab next to current tab

If this option is selected, when opening search results in a new tab, the new tab will be placed next to the current tab. Otherwise, it will be placed at the end.

You can temporarily reverse this setting by holding down the Control key when executing a search.

Use new window instead of new tab

If this option is selected, when opening search results in a new tab, a new foreground window will be used instead.

You can temporarily reverse this setting by holding down the Option key when executing a search.

Default Search Engine

Default search engine

Select the default search engine.

In PopSearch's search engine menu, preselect

This setting determines which search engine will be preselected when you invoke PopSearch. The options are as follows:

  • The default search engine
    If this option is selected, the default search engine will always be preselected in the search engine menu whenever you invoke PopSearch.
  • The last-used search engine (same for all sites)
    If this option is selected, PopSearch will remember which search engine you last used and will preselect it the next time you invoke PopSearch, ignoring the default search engine setting.
  • The last-used search engine (specific to each site)
    If this option is selected, PopSearch will keep a list of the 100 most recently visited sites and remember which search engine you last used on each of those sites. The next time you invoke PopSearch on one of these sites, the same search engine will be preselected. If a site is not yet in the recent sites list, the default search engine will be preselected.

Search Suggestions

This setting determines whether and what kind of a menu of search suggestions will be shown below the query box as you type a query. To select a suggestion, use the up/down arrow keys on the keyboard.

Show suggestions from Google

If this option is enabled, PopSearch will download and display under the query box a list of up to ten popular search queries from Google that are similar to the terms you have typed.

Note: The Google suggestions feature uses an unofficial API. It is likely that Google will discontinue this API in the future. When that happens, suggestions will stop working.

Show similar past search queries

If this option is enabled, PopSearch will keep a list of the 500 most recent search queries and use this list to display up to ten past queries containing your typed terms.

Don't show suggestions

If this option is enabled, PopSearch will not display suggestions under the query box.

Backup Service

PopSearch features the ability to back up your custom search engines to various web services, such as Delicious and Pinboard. If you have an account with one of these services, you can use the buttons at the bottom of the search engines page to back up your custom engines to the service, or to restore them once you've completed a backup. Backing up your engines is useful not only in case you lose your PopSearch settings for any reason, but also when you want to copy your engines from one computer to another.

The first time you back up your engines to a service, you will be prompted to enter your username and password for that service. Backing up and restoring take just a few seconds.

Note: PopSearch cannot back up to (or restore from) Delicious accounts that use a Yahoo login. In order to use back up your search engines to Delicious, please use a native Delicious account.

Character Encoding

Support non-Latin characters in search terms

Enabling this option will cause PopSearch to properly encode non-Latin (for example, Chinese) characters in search terms.

Note: Enabling this option may cause problems with some search engines when the search query includes extended-ASCII characters, such as characters with diacritical marks (e.g., á, ñ, or ü). If you enter extended-ASCII characters often, leave this option disabled.
