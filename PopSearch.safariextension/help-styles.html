Styles

Each PopSearch style is a CSS stylesheet with rules that control the appearance of the PopSearch popup. If you know CSS—and even if you don't—feel free to change whatever you like. We recommend editing a duplicate of a default style: that way, if you mess it up, you won't have to click Restore Defaults and lose all changes to all styles (including losing any new styles).

If you don't know CSS but would like to learn, a good starting point is HTML Dog's CSS Beginner Tutorial.

Have fun!
